# Project 1: CRM Sales Opportunities
Insight: This project analyzes B2B sales pipeline data from a fictitious computer hardware company. The dataset includes comprehensive details on accounts, products, sales teams, and sales opportunities.

Workflow:

**Imported and cleaned the raw data** using **Power Query** to ensure consistency.

**Established relational connections** between the five data files within the **Excel Data Model**. This relational structure significantly **improved the accuracy** and depth of the analysis.

Built **dynamic Pivot Tables** to answer key business questions, such as product win rates and sales team performance.

Created **clear, targeted visualizations** to transform complex data into actionable business insights.
